# Headshot Logic Documentation

This folder contains a simple educational demonstration of how hit detection works in games like Free Fire.

### What is in `headshot_logic.py`?
It's a Python script that calculates the distance between a "hit point" and a "head center". 
In game engines like Unity or Unreal (which Free Fire is built on), this is handled by:
1. **Raycasting**: Shooting an invisible line from the gun to see what it hits.
2. **Hitboxes**: Invisible shapes (like spheres or boxes) around character body parts.
3. **Bone Tracking**: Ensuring the hitbox moves perfectly with the character's head animation.
